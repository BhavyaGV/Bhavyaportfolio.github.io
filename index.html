Title: IOT based 3 in 1 Smart Meter

Description: IOT based 3 in 1 smart meter is an electronic device that records consumption of sources(Electricity,water and gas) andcommunicates the information to the supplier for monitoring and billing. This proposed system has 2 sections mainly,one is Home meter section and another one is Server section. Communication between these 2 sections is done through wireless network. The corresponding readings of all sources will be recorded in the server and also an app is provided for consumers and is continuously communicated to the controlling base section. The consumers can track the daily usage of all 3 sources and also the amount of consumption.
